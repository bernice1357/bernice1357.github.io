本篇會介紹 Vue 環境的安裝及建立！

Environment: Ubuntu

  • 安裝 npm(Node Package Manager) :::info 要確認裝的版本是 12 or 14以上（最新版本是 17） 用 14 就好了 :::
sudo apt-get install nodejs npm

確認有裝好：

npm -v
  • 安裝 Vue-cli
npm install -g @vue/cli

確認有裝好：

vue -v

:::danger
